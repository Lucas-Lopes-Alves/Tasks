# Tasks
A Linux terminal command to organize your tasks
---

Use the commands to organize your daily tasks
---

## 1.add
the add command let you add a new task, just use:
- `task add "task goes here"`

## 2.remove
the remove command let you remove a task based on its number, the number appear in other command called list, example:
- `task remove 1`

## 3.list
the list command lists all the tasks in the tasks file:
- `task list`

## 4.change
the change command change a task instead of removing it and adding it again:
- `tasks change 1 "new_task"`
